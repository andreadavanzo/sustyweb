Data Retention Policy

We don't want to hold onto more data than is otherwise necessary, which is why we set expiry tags to store information that is not being actively used so that it may be purged within 90 days. If data is being actively used, it will be verified that the information is still relevant to retain at 6 monthly intervals and if it is no longer useful to retain, that information will be removed within 30 days (with notice given to consumers).

Test passes if the policy mentioned is put into action and data with an expiration date is purged.
